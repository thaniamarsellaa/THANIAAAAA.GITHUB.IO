About Me

Nama saya Thania Marsella. Saya lahir di Pontianak, 23 Maret 2001 . Saya tinggal di Jl. Selat Panjang Gang Karya Mandiri Perumahan Residence Mandiri Blok A1, Kalimantan Barat. Jenis kelamin saya Perempuan. Saya beragama Kristen.

Pekerjaan saya saat ini adalah Sebagai Mahasiswa di Sekolah Tinggi Ilmu Ekonomi Indonesia Pontianak ( STIE Indonesia Pontianak ) dengan program studi S1 Manajemen.

Saya anak tunggal. Ayah saya bernama Tjie Khiong dan ibu saya bernama Lidya.
